# subway_analysis
지하철 데이터 분석 프로젝트

📌 프로젝트 개요 
이 프로젝트는 시간별 지하철 승하차 승객 수 데이터를 기반으로 근무지와 주거지를 구분하고, K-means 군집화를 통해 사고 패턴을 분류하는 것을 목표로 한다.
출근 지수와 퇴근 지수 같은 지표를 계산하여 근무지와 주거지를 구분하고, 군집별 출근지수와 퇴근지수의 특징을 도출하였다.

# 불필요한 컬럼,로우 제거

MySQL을 이용해 '사용월' 컬럼과, '작업일자' 컬럼을 제거하였고, 출퇴근 시간의 정보만 필요했기 때문에 위에서 언급한 컬럼을 추출하였다. 장기간, 넓은 지역의 데이터를 분석할 필요 없이 원하는 인사이트를 도출할 수 있다고 판단해 2025년 11월 데이터 그리고 1,2,3호선의 데이터만 추출하였다.

# 파생변수 생성

출근지수 = 08-09시 하차인원/18-19시 승차인원
퇴근지수 = 18-19시 승차인원/08-09시 하차인원

📊 주요 분석 내용

1️⃣ 출/퇴근시간 혼잡지역 분석
08-09시 하차 승객 수 TOP10 지역 추출
18-19시 승차 승객 수 TOP10 지역 추출
출근지수가 높은 순서대로 지하철역 정렬
퇴근지수가 높은 순서대로 지하철역 정렬

2️⃣ K-Means 군집화
사용된 변수: 출근지수, 퇴근지수
최적 K값 결정 방법 (Elbow Method)
군집별 출근지수와 퇴근지수의 평균 지표

3️⃣ 시각화
출근시간 하차 승객 수 TOP10 barplot
퇴근시간 승차 승객 수 TOP10 barplot
Elbow Method로 군집 수를 찾는 lineplot
군집별로 출근지수와 퇴근지수의 분포를 찾기 위한 scatterplot

🧠 분석 결과 요약
🔹 Cluster 0 — (주거지)
특징 요약 : 출근지수가 1보다 작고 퇴근지수가 1보다 큼
🔹 Cluster 1 — (업무지)
특징 요약 : 출근지수가 1보다 크고 퇴근지수가 1보다 작음


📝 결론
본 프로젝트를 통해 근무지가 몰려 있는 지역과 주거지가 몰려 있는 지역을 정량적으로 파악하였으며,
K-means 군집화를 통해 주거지와 거주지의 출근지수/퇴근지수 분포를 2개의 군집으로 분류하여 특성을 파악 할 수 있었다.
